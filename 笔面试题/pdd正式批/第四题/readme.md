## 题目
情景题，给定一个数组，每个元素的值代表该元素的类型，再给定一个数字`m`，表示能将数组分为`m`个区域，每个区域需要连续，请求得每个区域的类型数量之和

如：数组为`[5,5,6,6,5,6,1,5]`,`m`为3，则其中一种分区方案为`[5,5,6],[5,5,6],[1,5]`,最终和为`2+2+2=6`

## 思路
这里我无根据的采用了贪心策略，遍历整个数组，每次遇到相同并连续的两个值时，将其分区，最终得到总和

其实这里有一种很明显的错误比如上述案例第二个分区的第一个5改成6那么按照我的思路是`[5],[5,6],[6,5,6,1,5]`最终是`1+2+3=6`，而很明显可以这样`[5,5,6],[6,5],[6,1,5]`结果是`2+2+3=7`
